Política de Privacidade

A sua privacidade é importante para nós. Esta política explica como o aplicativo Pode Parar Aqui lida com as suas informações.

1. Coleta de Localização

O aplicativo utiliza a sua localização geográfica via GPS apenas enquanto o app está em uso. Essa informação é necessária para identificar onde as vagas de estacionamento estão sendo indicadas e para mostrar as vagas próximas a você.

2. Dados Pessoais

O aplicativo não exige a criação de conta, nome, e-mail ou qualquer dado pessoal identificável para o uso das funções básicas de indicação e busca de vagas.

3. Compartilhamento de Dados

As coordenadas geográficas das vagas e a quantidade de vagas informadas são compartilhadas publicamente com outros usuários do aplicativo para fins exclusivos de auxílio à mobilidade urbana. Não compartilhamos dados que identifiquem o dispositivo do usuário.

4. Segurança

Os dados das vagas são armazenados de forma segura no Firebase (Google Cloud), visando a integridade das informações.

Última atualização: Janeiro de 2026.
